# 座位管理系統-前端

- 設計發想：每次學生考試都需重新隨機排定梅花座，若學生數過多，這項工作變得耗時耗力
- 使用者：助教、教授
- 系統功能：只需設定教室和學生名單，即可取得排好的隨機座位表。
